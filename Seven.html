Can I borrow your ...?

Exercises

  1. Take a look at the file /Scripts/itemsonloan/itemsonloan.js, and see how the itemsOnLoan module has a dependency to both ngRoute and itemsOnLoanFilters. The itemsOnLoanFilters module can be found in /Scripts/itemsonloan/filters.js. Can you see where on the page the custom filter created in this module is used?

  2. Create another custom filter called numOnLoan in filters.js, which returns the number of items with the status 'On Loan'.

  3. Add the logic needed in order to display the number of items on loan in the "Number of items on loan" column in the table to the left.

  4. Click on the details links next to Sue. Right now all the borrowed items and the items on loan are shown in both boxes. Can you make the borrowed items only display in the first box, and the items on loan only display in the second?

  5. Can you make the Borrowed and On Loan boxes in the details view only display if they contain items?

More information

What we're using here is the ngView directive, the ngShow directive, the routeProvider service and custom filters.
