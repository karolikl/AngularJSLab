Exercises

  1. Have a look at the javascript files in the /scripts/shoppinglist folder. Make the item name and the item amount show in the table to the left.

  2. Notice that the ShoppingList controller contains a function called "add". Make sure this function is called when the submit button is clicked.

  3. Notice that the ShoppingList controller contains a function called "delete". Make sure this function is called when the delete link in the last column is clicked.

  4. Add another link to the last column called "Add one". This link should call a function in the ShoppingList controller called "addOne", which adds 1 to the number of items of the item called.

More information

What we're using here is the ngApp directive and the ngClick directive.
