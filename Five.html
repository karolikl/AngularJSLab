Exercises

  1. Make lastname a required field, displaying an error message below the input if it is invalid.

  2. Phone number should not be required, but it has to contain between 8 and 12 characters. An error message should be displayed under the input if the length requirement is not satisfied. Please add validation logic to satisfy these requirements.

  3. The value entered into the Website input has to be a valid url. Can you add an error message below the input if the value entered is not valid?

  4. If the form is invalid, the submit button should be disabled. Can you fix this?

  5. At the moment the text "The form has been changed" is showing at all times. Can you make this text display only when the form is dirty (has been changed)?

More information

What we're using here is the ngShow directive, the input.text directive, the input.url directive and the ngDisabled directive.
