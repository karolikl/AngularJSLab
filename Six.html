Contacts on a single page

  • Table view
  • List view

Exercises

  1. Open the developer toolbar in your browser and watch the network traffic as you change back and forth between Table view and List view.

  2. Create another ng-template script tag on the page with the id plaintext.html. Within this template, loop through the array of friends and output the name and email in plain text. Note: If you run this project on a webserver, the template can be a separate html file. However, if it's in a separate file and not running on a webserver, the browser will give you a cross origin request error. Putting the template in a script tag on the same page as we do here results in AngularJS retrieving the template without doing an AJAX request, avoiding all cross origin request errors.

  3. Add a route to /Scripts/contactlist/contactlist.js, pointing to the template previously created, and the ContactListCtrl controller.

  4. Add a "Plain text view" tab to the page, and test if it works as it should. Make sure the name search also works for your template.

More information

What we're using here is the ngView directive and the routeProvider service.
