Exercises

  1. Take a look at the following file: /scripts/movies/movieCtrl.js, and see how an array of movies is created on the scope. Notice that the length of the movie is not dispayed in the overview, can you fix this?

  2. Each movie also contains an array of stars. Using the ng-repeat attribute, can you display the names of all the stars for each movie?

  3. The movie budget is now shown as a simple number, and it would be better for this to be displayed with a currency sign and proper formatting. Use the currency filter in AngularJS to improve how budget is displayed.

  4. Using another filter, make the title of the movie uppercased.

More information

What we're using here are the ngController directive, the ngRepeat directive, the currency filter and the the uppercase filter.
