The staring dogs

Creating a clickable image

Exercises

  1. Have a look at the files in the /Images folder. Edit the img element on this page so that it displays the correct image based on the selected value of the "Dog size" dropdown list.

  2. Create another dropdown list called "Breed" which binds to the breed property of the objects in the dogs array.

  3. Create an anchor tag around the image, containing the link to a Google search for the breed selected in the "Breed" dropdown. Hint: The direct url to a Google search is https://www.google.com/#q=yourQuery

More information

What we're using here is the ngApp directive, the ngController directive, the ngSrc directive and the ngHref directive.
